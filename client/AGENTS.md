# Glass Hero Hub - AGENTS.md

## Project Overview

This project is a sleek, modern web application for the Overwatch Account Share platform, built with React and Vite. It features a stunning glassmorphism design, providing a visually appealing and intuitive user experience.

## Core Technologies

-   **React**: A JavaScript library for building user interfaces.
-   **Vite**: A fast build tool and development server for modern web projects.
-   **TypeScript**: A typed superset of JavaScript that compiles to plain JavaScript.
-   **Tailwind CSS**: A utility-first CSS framework for rapid UI development.
-   **Framer Motion**: A library for creating animations and gestures.
-   **React Router**: A library for handling routing in a React application.

## Build and Test Commands

-   **Run development server**: `npm run dev`
-   **Build for production**: `npm run build`
-   **Lint code**: `npm run lint`
-   **Preview production build**: `npm run preview`

## Code Style Guidelines

-   The code follows standard React and TypeScript conventions, enforced by ESLint. The configuration can be found in `eslint.config.js`.
-   Code is formatted using Prettier.
